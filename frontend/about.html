Source of Data

All exchange rate data shown in this application is sourced from the U.S. Department of the Treasury. Their open fiscal API provides current and historical foreign exchange rates published quarterly.

Purpose

This platform was developed to make U.S. bond-related currency data more accessible, visual, and usable for financial analysts, students, and the general public. It was built using modern web technologies with a focus on simplicity and accuracy.
